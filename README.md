# odin-recipes

This is a simple project that aims to demonstrate basic HTML knowledge acquired in The Odin Project's `HTML Foundations` section in the `Foundations` path. It showcases the following HTML features:

* Elements and Tags
* HTML Boilerplate
* Text
* Lists
   * Ordered lists
   * Unordered lists
* Links
* Images

Note: this is a very basic implementation and will likely be revisited in the future to add new features (such as CSS).
